# axialslider

Simple slider with two axes and buttons

## Installation

    npm i axialslider

## Usage

    import axialslider from 'axialslider';

    window.addEventListener('load', function () {
        axialslider({
          class: 'axialslider', // CSS class name, default: axialslider
          color: '#19bd9a' // Also you can use rgba
        });
    });
